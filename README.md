# Install and start project.
Skeleton for new project by Symfony + ReactJS.

## Required software
1) PHP 8 +
2) Node 14 +
3) YARN 
4) Symfony CLI

## Install project
1) Clone project from git repository;
   
2) Run commands for install all library:
   * npm install
   * yarn install
   * composer install

## Run project
Start commands:
+ symfony server:start --no-tls
+ yarn encore dev --watch
